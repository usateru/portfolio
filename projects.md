## 2. メール自動処理ワークフロー（Apache Camel）
**技術:** Java（Camel）、IMAP、JSON API、Docker  
**期間:** 2024〜2025  

### 概要
特定アカウントのメール受信をトリガーに、  
JSON 生成〜Web API 連携までを自動化するバックグラウンドアプリを構築。

### 役割
- Camel ルート設計  
- テンプレート／CSV を含む ZIP の検証  
- Web API 呼び出しとエラーハンドリング  
- 運用容易性を考慮した分岐フロー設計
